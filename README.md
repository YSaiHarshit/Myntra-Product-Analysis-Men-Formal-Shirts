# Myntra Product Analysis – Men Formal Shirts

## Project Overview

This project analyzes **Men Formal Shirts** available on Myntra using **Web Scraping and Exploratory Data Analysis (EDA)**. Product data was collected from the Myntra website using Selenium and analyzed to understand pricing patterns, discounts, customer ratings, reviews, and brand performance.

## Objectives

* Analyze pricing trends of formal shirts.
* Study discount patterns and promotional strategies.
* Evaluate customer ratings and reviews.
* Identify top-performing brands.
* Generate business insights from product data.

## Technologies Used

* Python
* Selenium
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Dataset Features

The dataset contains:

* Brand
* Product Name
* Price
* Old Price
* Discount
* Rating
* Reviews

## Data Analysis Performed

* Data Cleaning and Preprocessing
* Price Distribution Analysis
* Brand-wise Product Analysis
* Discount Distribution Analysis
* Rating Distribution Analysis
* Reviews Analysis
* Correlation Analysis

## Key Insights

* Most formal shirts are priced between ₹500 and ₹1100.
* Discounts between 50% and 60% are the most common.
* Park Avenue and Raymond dominate the category.
* Most products have ratings above 4.0.
* Customer engagement is concentrated on a few popular products.

## Conclusion

The analysis shows that customers prefer affordable, highly rated, and discounted formal shirts. Pricing, discounts, ratings, reviews, and brand reputation play a significant role in influencing customer purchase decisions.
